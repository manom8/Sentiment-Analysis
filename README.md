# Comparative Sentiment Analysis on Twitter Data

## Overview
This project performs sentiment analysis on Twitter data using multiple machine learning models. The objective is to compare model performance in classifying tweets as positive or negative.

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Feedforward Neural Network (FFN)

## Technologies
- Python
- scikit-learn
- Pandas
- NumPy

## Key Steps
- Data preprocessing and cleaning
- Text vectorization
- Model training and evaluation
- Performance comparison using classification metrics

## Results
The models were evaluated using accuracy and other classification metrics to determine the most effective approach for sentiment classification.
